# Codes

This repository contains various scripts, codes, and libraries implemented by the owner during their college years. The following is a brief description of the directories present in this repository:

## Algorithms

This directory contains implementations of various algorithms that the owner learned during their college studies. It includes sorting algorithms, searching algorithms, graph algorithms, and more.

## Coding-Problems

This directory contains solutions to various coding problems that the owner solved during their college studies. These problems include both theoretical and practical coding problems that are usually given to students in programming courses.

## Libraries

This directory contains the owner's own implementations of C and Java libraries. These libraries can be used in other projects, and they serve as a learning exercise for the owner to understand how these libraries work under the hood.

## Makefile-Templates

This directory contains Makefile templates for single and multiple-file dependency scripts. Makefiles are often used to automate the build process for complex programs and projects, and these templates can serve as a starting point for creating new Makefiles.

## Misc

This directory contains miscellaneous scripts that don't fit into any other directory. These scripts may be useful for other projects or may serve as examples for various programming concepts.

## Runtime-Tests

This directory contains scripts used for benchmarking and measuring the runtime of various programs implemented by the owner.

## Terminal-Cursor-Codes

This directory contains codes for manipulating the terminal cursor. These codes can be used to implement various features in terminal-based applications.

## VS-Code-Customizations

This directory contains a single snippets file that can be imported into VS Code to add custom code snippets to the editor. These snippets can save time when writing code by providing pre-written templates for common programming tasks.
